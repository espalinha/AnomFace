# AnomFace
## Introduction
"This repository is the result of my undergraduate research project and aims to develop an efficient and high-accuracy algorithm for facial recognition and anonymization.

I intend to use JAX to implement my own DenseNet architecture. In addition, I plan to apply optimizations to reduce the model’s computational cost and enhance performance."
